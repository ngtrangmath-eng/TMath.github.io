# Cac lenh tao phieu

## Lenh mac dinh

```bash
node Phieu-hoc-tap-mau/main.js
```

Doc:

```text
Phieu-hoc-tap-mau/bai-hoc/bai-hoc-mau.json
```

Ghi file theo `lesson.output`, neu khong co se ghi vao:

```text
Phieu-hoc-tap-mau/output/[Ma-bai].html
```

## Lenh tao tu file bai hoc khac

```bash
node Phieu-hoc-tap-mau/main.js Phieu-hoc-tap-mau/bai-hoc/ten-bai.json
```

## Lenh chi dinh file dau ra

```bash
node Phieu-hoc-tap-mau/main.js Phieu-hoc-tap-mau/bai-hoc/ten-bai.json "Toan 6/Toan6-Chuong1/Toan6-Chuong1-Bai8/Toan6-Chuong1-Bai8.html"
```

## Lenh kiem tra cu phap script

```bash
node --check Phieu-hoc-tap-mau/main.js
```

## Mau cau lenh cho agent

```text
Hay tao JSON bai hoc moi theo Phieu-hoc-tap-mau/bai-hoc/bai-hoc-mau.json, sau do chay main.js de sinh HTML.
```

```text
Ap noi dung Bai8-PhanA.md, Bai8-PhanB.md, Bai8-PhanC.md vao Phieu-hoc-tap-mau va tao phieu hoc tap hoan chinh.
```
